# ====================MongoDB Shell使用說明===================
# 1. 要先安裝cli工具, 這是官方連結: https://www.mongodb.com/docs/mongodb-shell/install/#procedure
# 2. 連上mongoDB, 語法類似以下, 要把pem檔路徑改成自己放pem的路徑
mongosh "mongodb+srv://cluster0.edk0n6b.mongodb.net/?authSource=%24external&authMechanism=MONGODB-X509" --apiVersion 1 --tls --tlsCertificateKeyFile "D:\HeroFishingServer\Keys\mongodb_X509-cert-6644120643259731564.pem"
# 3. 連上後就可以進行CRUD等操作, 可參考官方文件: https://www.mongodb.com/docs/mongodb-shell/crud
# 4. 測試mongoShell可以修改MongoShellTest.js搭配以下指令來執行
mongosh "mongodb+srv://cluster0.edk0n6b.mongodb.net/?authSource=%24external&authMechanism=MONGODB-X509" --apiVersion 1 --tls --tlsCertificateKeyFile "D:\HeroFishingServer\Keys\mongodb_X509-cert-6644120643259731564.pem" -f D:\HeroFishingServer\Realm\MongoShellTest.js
